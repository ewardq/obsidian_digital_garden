Concatenate files and print on the standard output.
With no FILE, or when FILE is -, read standard input.

> [! Tip] Syntax
	cat [OPTION]... [FILE]...


Output `f`'s contents, then standard input, then `g`'s contents.
```bash
cat f - g
```

Copy standard input to standard output.
```bash
cat
```
