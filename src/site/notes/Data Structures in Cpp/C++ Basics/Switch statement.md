The switch statement is a decision making statement available in C that can be uses instead of several linked `if ... else` statements.
## Switch syntax

```c++
switch( expression ){
	case value1:
		//statement_1;
		//statement_2;
		//statement_n;
		break;
	case value2:
		//statement_1;
		//statement_2;
		//statement_n;
		break;
	case value3:
		//statement_1;
		//statement_2;
		//statement_n;
		break;
	default:
		//statement_1;
		//statement_n;
}
```
