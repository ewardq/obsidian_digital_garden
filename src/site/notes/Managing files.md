---
How to create files?

Using [[touch\|touch]]
```bash
touch newFile
```

Using data streams `>>`
```bash
> newFile
```

---
How to rename files/directories?

Using [[mv\|mv]]
```bash
mv old_name new_name
```

```bash
mv old_directory_name new_directory_name
```

---
How to remove files?

Using [[rm\|rm]]
```bash
rm file
```
